#This is a small project on sentiment analysis on an IMDB dataset using some simple neural networks.
The goal of sentiment analysis is to label a sentence as positive or negative.
The IMDB dataset contains 50,000 records of English sentences. The sentences are labeled either positive or negative.
![alt text](image.png)

Simple neural network models include single-layer networks, convolutional networks, LSTMs, convolutional networks combined with LSTMs, BLSTMs, and transformers.
